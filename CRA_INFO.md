# Configuração do Create React App

Este projeto foi iniciado com [Create React App](https://github.com/facebook/create-react-app).

## Scripts Disponíveis

### `npm start`

Inicia o app em modo de desenvolvimento.
Abra [http://localhost:3000](http://localhost:3000) para ver no navegador.

A página recarrega quando você faz alterações.
Você verá erros de lint no console.

### `npm test`

Inicia o test runner em modo watch interativo.
Veja a seção sobre [running tests](https://facebook.github.io/create-react-app/docs/running-tests) para mais informações.

### `npm run build`

Compila o app para produção na pasta `build`.
Otimiza o React para melhor performance.

O build é minificado e os nomes de arquivos incluem hashes.
Seu app está pronto para deploy!

### `npm run eject`

**Nota: esta é uma operação sem volta. Uma vez que você `eject`, não pode voltar!**

Se você não está satisfeito com a ferramenta de build e escolhas de configuração, você pode `eject` a qualquer momento.
